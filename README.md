# 📢 Telegram Mention Bot

Бот для упоминания участников группы с разбивкой по категориям (мужчины, женщины или любые произвольные группы).

---

## 🚀 Установка и запуск

### 1. Получить токен бота
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot` и следуй инструкциям
3. Скопируй полученный **токен**

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Запустить бота
```bash
# Linux / macOS
BOT_TOKEN="ваш_токен_сюда" python bot.py

# Windows (PowerShell)
$env:BOT_TOKEN="ваш_токен_сюда"; python bot.py

# Windows (cmd)
set BOT_TOKEN=ваш_токен_сюда && python bot.py
```

### 4. Добавить бота в группу
- Добавь бота в Telegram-группу
- Дай ему права **администратора** (чтобы видеть сообщения)

---

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `/all` | Отметить всех участников в базе |
| `/men` | Отметить всех из группы «men» |
| `/women` | Отметить всех из группы «women» |
| `/group <название>` | Отметить произвольную группу |
| `/addme men` | Добавить себя в группу «Мужчины» |
| `/addme women` | Добавить себя в группу «Женщины» |
| `/addme <название>` | Добавить себя в любую группу |
| `/adduser @username <группа>` | Добавить другого (только для админов) |
| `/removeme` | Убрать себя из всех групп |
| `/mygroups` | Посмотреть свои группы |
| `/groups` | Все группы в чате со счётчиком |
| `/members <название>` | Список участников группы |

---

## 💡 Как это работает

1. Каждый участник сам добавляет себя командой `/addme men` или `/addme women`
2. Администраторы могут добавлять других через `/adduser`
3. Данные хранятся в локальном SQLite файле `bot.db`
4. Один бот работает сразу во **многих группах** — данные разделены по `chat_id`

## ☁️ Деплой (опционально)

Для постоянной работы можно запустить на сервере через `systemd` или `screen`:

```bash
# screen
screen -S tgbot
BOT_TOKEN="токен" python bot.py
# Ctrl+A+D чтобы отсоединиться
```

Или создай `.env` файл и используй `python-dotenv` для удобства.
